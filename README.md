# quarto-dashboards

This repository contains code and data to support building a Quarto dashboard that replicates the [NC Child Data Center Dashboard](https://ncchild.org/data-center/). You can [see what it will look like here](https://projects.indicatrix.org/quarto-dashboards/).

## Usage in RStudio

Download this repository by clicking the green "Code" button above and choosing "Download ZIP." Unzip it and open `nc-child-quarto.Rproj` in RStudio, and open `dashboard_exercise.qmd` in the files pane to get started.

## Usage on Posit Cloud

To use on [Posit Cloud](https://posit.cloud), choose New Project > New Project from Git Repository, and then enter <https://github.com/mattwigway/quarto-dashboards>.
